## Example

## Installation

FlexiblePathMenu is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "FlexiblePathMenu", :git=> 'https://github.com/matrixs/FlexiblePathMenu.git'
```
